Usage
=====

Installation
------------

To use BigFan, first clone the repo in github:

using ssh:

.. code-block:: console

  (.venv) $ git clone git@github.com:millerannalise/BigFan.git
using https:

.. code-block:: console

  (.venv) $ git clone https://github.com/millerannalise/BigFan.git
